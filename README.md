# django
Django Tutorial

## 0. Prerequisites
Python installed

## 1. Create env
```
python -m venv .venv
source .venv/bin/activate
```

## 2. Install django
```
pip3 install django
```

## 3. Create project
```
django-admin startproject config .
```

## 4. Create app
```
./manage.py startapp app
```

## 5. Make migration app
```
./manage.py migrate 
```

## 10. Run server
```
./manage.py runserver
```
